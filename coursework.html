Simplified Technical English Dictionary

This website can be used to search the dictionary of the Simplified Technical English standard.

The Simplified Technical English (STE) standard is a controlled language for use in English technical documentation.

The dictionary of the standard shows words which are approved and non-approved according to the standard. Alternatives to non-approved words and definitions for words used in the standard are also shown.

For more information about the Simplified Technical English standard, see the Simplified Technical English Management Group website

Hover over a heading in the table below for a detailed description of the information.
